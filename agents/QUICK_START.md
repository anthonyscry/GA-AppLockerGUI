## Multi-Agent Workflow

For complex tasks, use multiple agents:

1. **Start with Project Lead** - Sets direction
2. **Add specialists** - Each handles their domain
3. **Agents coordinate** - Through file changes and reports

Example: Security Review
```
1. Project Lead (orchestrates)
2. Security Analyst (finds issues)
3. Code Validator (verifies fixes)
4. QA Engineer (tests fixes)
```
